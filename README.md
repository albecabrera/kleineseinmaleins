# Kleines Einmaleins

Printable multiplication table worksheet generator for primary school students.

**[Live Demo](https://albecabrera.github.io/kleineseinmaleins/)**

## What it does

- Picks 5 random multiplication tables (1–10) on each run
- Shuffles the factors so every worksheet is unique
- Includes name / date / class fields and a score box per column
- Print-ready A4 layout — no margins, portrait orientation

## Use

Open `index.html` in a browser or visit the live demo.

| Button | Action |
|--------|--------|
| 🔀 Neu generieren | Generate a new random worksheet |
| 🖨️ Drucken / PDF | Print or save as PDF |

Print settings: portrait, A4, no margins.

## Stack

Single HTML file — no dependencies, no build step.
